VuLink SDI-12 Setup

VuLink is a telemetry device that works with all kinds of water monitoring instruments. Its job is to upload data from those instruments to the cloud. There are different ways VuLink can get that data from an instrument. One of those ways is called SDI-12. That's a communication protocol, which just means it's a bunch of rules about the order in which bits and bytes must move from one device to another.

This project features the SDI-12 setup portion of the VuSitu mobile app. It's simple because the UX and software teams worked hard to make it that way. I wrote all of the text for this feature.
