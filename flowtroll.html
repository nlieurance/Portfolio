FlowTROLL Help Text

I wrote this help text for some velocity sensors originally made by an Australian company. In-Situ redesigned the sensors so a user could set them up with the VuSitu mobile app. That's an improved user experience. But some of the advanced settings still needed a bit of explanation.

The messages I came up with aren't short, but they provide important details. The simple vocabulary and use of examples make this text easy to understand.
